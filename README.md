## كيفية الاستخدام

### محلياً
افتح `index.html` في أي متصفح حديث (Chrome / Edge / Firefox)

### على GitHub Pages
1. ارفع جميع الملفات في مجلد واحد
2. فعّل GitHub Pages من Settings → Pages
3. اختر `main` branch و `/ (root)` folder
4. الرابط سيكون: `https://username.github.io/repo-name/`
